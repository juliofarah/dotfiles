###leader key and bindings
leader key is mapped to ``[space]`` to allow ``[space] + w`` and ``[space] + q`` to be used as ``:w`` and ``:q``, respectively.

``[ctrl] + <h / j / k / l>`` to jump between panes.

``[space] + r`` to reload vim configurations.
